# Awesome Satisfactory [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> A curated list of awesome things related to Satisfactory - the factory-building simulation game by Coffee Stain Studios

Satisfactory is a first-person factory building game with a dash of exploration and combat. Build massive factories, automate production lines, and optimize your layouts on an alien planet.
